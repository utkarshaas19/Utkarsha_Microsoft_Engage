AVISHKAR - The Face Detection System

Prerequisites to Execute the Project :
1. Python 3.6 or above installed in your device : https://www.python.org/downloads/
2. Windows 7 or above
3. Python packages installed : numPy, keras, tensorflow, PILLOW, tkinter, imutils, cv2, cmake, pyttsx3, mediapipe, argparse
   Use : pip install packagename

Steps to Run the Project :
1. Execute the Utkarsha.py file first, it is the homescreen of the application
   Click buttons and access all other modules implemented in it
2. For Object detection, open the terminal and execute the command : python objectdetection.py --prototxt MobileNetSSD_deploy.prototxt.txt --model MobileNetSSD_deploy.caffemodel
3. For Quitting from a module : press key 'q' until the screen gets still, and then press the close button on the right hand side top of the frame
4. For Quitting from the homepage : Press the quit button
